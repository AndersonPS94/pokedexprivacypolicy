Política de Privacidade

Aplicativo Pokédex

Introdução

Última Atualização: 17 de Novembro de 2025

Esta Política de Privacidade descreve como o aplicativo móvel Pokédex (o "Aplicativo"), desenvolvido por AndersonPS (o "Desenvolvedor"), trata as informações de seus usuários (cada um, um "Usuário").

O Desenvolvedor está profundamente comprometido em proteger a privacidade de seus Usuários. Esta política foi elaborada para ser **clara, objetiva e transparente**, garantindo total conformidade com as políticas de **Dados do Usuário** do Google Play.

1. Coleta de Dados Pessoais

O Aplicativo Pokédex **NÃO coleta nenhuma informação de identificação pessoal (PII) ou dados pessoais** de seus Usuários.

O Aplicativo foi projetado para funcionar sem a necessidade de coletar, transmitir ou armazenar qualquer dado que possa identificar um indivíduo, como nome, endereço de e-mail, localização precisa, identificadores de dispositivo persistentes ou qualquer outra informação sensível.

2. Uso de Dados

O Aplicativo Pokédex tem como única finalidade fornecer informações sobre Pokémons. Para isso, ele realiza as seguintes operações:

  • **Consumo de API Pública:** O Aplicativo consome dados exclusivamente de uma **API pública** (Application Programming Interface) para listar e exibir informações sobre Pokémons. Esses dados são públicos e não estão vinculados a nenhum Usuário.
  • **Cache Local:** O Aplicativo salva dados (informações de Pokémons) **apenas localmente no dispositivo do Usuário** utilizando um Room Database para fins de cache e para permitir o funcionamento offline. Esses dados de cache são estritamente operacionais e não são transmitidos para o Desenvolvedor ou terceiros.

Nenhum dado de Usuário é usado para fins de publicidade, análise, rastreamento ou qualquer outra finalidade comercial.

3. Compartilhamento de Dados com Terceiros

O Aplicativo Pokédex **NÃO compartilha NENHUM dado de Usuário com terceiros.**

Dado que o Aplicativo não coleta dados pessoais, não há dados a serem compartilhados. O Aplicativo não utiliza serviços de terceiros como Firebase, Google Analytics, SDKs de publicidade ou qualquer outro serviço de rastreamento.

4. Retenção de Dados

Os dados (informações de Pokémons) armazenados localmente no dispositivo do Usuário para fins de cache são retidos apenas enquanto o Aplicativo estiver instalado no dispositivo. O Desenvolvedor não tem acesso a esses dados e, portanto, não os retém.

5. Segurança dos Dados

Embora o Aplicativo não colete dados pessoais, o Desenvolvedor adota medidas razoáveis para garantir a segurança dos dados operacionais (cache de Pokémons) armazenados localmente. O Aplicativo não exige permissões sensíveis do Android, minimizando o risco de acesso não autorizado a informações do dispositivo.

6. Privacidade de Menores

O Aplicativo Pokédex **NÃO é voltado para crianças** menores de 13 anos. No entanto, devido à natureza do Aplicativo (que não coleta dados pessoais), a privacidade de qualquer Usuário, independentemente da idade, é protegida pelo fato de que nenhuma informação pessoal é coletada.

7. Alterações a Esta Política de Privacidade

O Desenvolvedor reserva-se o direito de atualizar esta Política de Privacidade a qualquer momento. Quando o fizer, a data de "Última Atualização" no topo desta página será revisada. Encorajamos os Usuários a verificar esta página periodicamente para quaisquer alterações. O uso continuado do Aplicativo após a publicação de alterações nesta política será considerado como aceitação dessas alterações.

Declaração de Conformidade com o Google Play

O Aplicativo Pokédex está em total conformidade com a Política de Dados do Usuário do Google Play, pois **não coleta, armazena ou compartilha dados pessoais ou confidenciais do Usuário**.
